Habito — Habit Tracker (MERN Stack)

Habito is a modern habit tracking web application built using the MERN stack (MongoDB, Express, React, Node.js).
The application helps users build consistency by tracking daily habits, visualizing progress, and maintaining streaks.

This project was developed as the Final Capstone Project for the MERN Full Stack Development Course.

---

 Features

🔐 Authentication

- User Signup
- User Login
- JWT Authentication
- Secure user sessions

📝 Habit Management

- Add new habits
- Mark habits as completed
- Delete habits
- Habit persistence using MongoDB

📊 Statistics Dashboard

- Weekly habit completion graph
- Monthly habit tracking
- GitHub-style activity heatmap
- Habit completion insights

📈 Productivity Tracking

- Habit streak tracking
- Daily progress ring
- Completion rate statistics

👤 Profile

- User profile overview
- Habit statistics
- Best streak tracking
- Profile avatar upload

🎨 Modern UI

- Built with React + Tailwind CSS
- Orange themed productivity interface
- Responsive layout
- Mobile bottom navigation
